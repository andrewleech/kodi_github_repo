Welcome to the alelec kodi repository

This is a repository server for Kodi (http://kodi.tv/)
To use this repo, download the zip file for repository.alelec and install it into kodi by normal "Install from ZIP" method

The server is designed to automatically serve addons directly from github, parsing for version tags and building releases on demand.
See https://github.com/andrewleech/kodi_github_repo for more info about the server.
